4. Быстрый старт

Как создать новое размещение с нуля

  1. Перейдите в раздел Типы размещения → создайте категорию (например, "Домик").
  2. Перейдите в раздел Размещения → нажмите "+ Добавить Accommodation".
  3. Заполните поля: Название, Описание, Тип размещения, Вместимость, Цена за ночь.
  4. Загрузите главное изображение (обложку) и галерею фотографий.
  5. Установите галочки Активно и при необходимости Показывать на главной.
  6. Нажмите Сохранить.

Как обработать новое бронирование

  1. Раздел Бронирование — новая бронь появляется со статусом pending.
  2. Откройте бронь на редактирование (иконка ✏️).
  3. Проверьте данные клиента, даты, объект.
  4. Измените Статус на confirmed (подтверждено) или cancelled (отменено).
  5. Добавьте примечания при необходимости.
  6. Сохраните.

Как обновить прайс-лист

  1. Подготовьте Excel-файл с актуальными ценами.
  2. Перейдите в Прайс-лист → откройте запись на редактирование.
  3. Загрузите новый файл в поле Excel файл.
  4. Сохраните — система автоматически распарсит данные в JSON.
Важно: Формат Excel-файла должен соответствовать ожидаемой структуре. При изменении структуры прайс-листа свяжитесь с разработчиком.

10. Галерея апартаментов

Управление фотографиями для всех объектов размещения. Загружайте фото к конкретным объектам, используйте массовую загрузку и задавайте порядок отображения.

11. Особенности размещения

Дополнительные характеристики объектов: наличие балкона, вид на озеро, дровяная печь и т.д.

11.1 Шаблоны особенностей

Готовые наборы характеристик для быстрого применения к нескольким объектам. Создайте шаблон один раз и используйте многократно.

11.2 Пункты шаблона

Отдельные элементы (пункты) внутри шаблона. Например, шаблон "Комфорт" может содержать пункты: "Кондиционер", "Тёплые полы", "Smart TV".

11.3 Применить особенности

Массовое применение шаблонов характеристик к выбранным объектам размещения. Выберите шаблон и объекты — система присвоит все пункты за один раз.

14. Прайс-лист

Управление ценами через загрузку Excel-файла или редактирование JSON-данных.

15. Юридические страницы

Управление правовыми документами: договор оферты, политика конфиденциальности, правила проживания. Контент редактируется через Rich Text редактор.

16. Контакты

16.1 Телефоны

Поля: Номер (+375 XX XXX-XX-XX), Описание ("Ресепшн 24/7"), Порядок, Активно.

16.2 Email адреса

Аналогично телефонам — управление email для разных целей (бронирование, общие вопросы, жалобы).

17. Отзывы

Колонки: ID, Имя, Email, Рейтинг (1-5), Текст, Одобрен, Создан.

Модерация: Новые отзывы не появляются на сайте до одобрения администратором. Проверяйте раздел регулярно.

18. Почта

18.1 SMTP настройки

Конфигурация почтового сервера: хост, порт, логин, пароль, шифрование (SSL/TLS). Отвечает за отправку всех писем с сайта (подтверждения бронирования, уведомления).

Важно: Не изменяйте SMTP-настройки без консультации с разработчиком. Неправильная конфигурация приведет к недоставке писем.

18.2 Шаблоны писем

Управление шаблонами email-уведомлений: подтверждение бронирования, напоминание о заезде, благодарность за отзыв. Поддерживаются переменные (например, {customerName}, {bookingId}).

18.3 Отправленные письма

История отправленных писем: получатель, тема, статус доставки, дата отправки. Полезно для диагностики проблем с доставкой.

19. Слайдер

19.1 Галерея

19.2 Слайдер О нас

Фотографии для страницы "О нас". Колонки: ID, Заголовок, Изображение, Порядок сортировки, Активен, Создано.

20. Администраторы

Колонки: ID, Email, Имя, Роль, Активен, Последний вход.

Безопасность: Не создавайте учетные записи с правами superadmin без крайней необходимости. Для контент-менеджеров используйте роль editor.

22. Работа с Rich Text (WYSIWYG-редактор)

Поля "Описание" используют визуальный редактор:

  • Отменить / Повторить
  • B — жирный
  • I — курсив
  • Подчеркивание / маркер
  • Выравнивание (по левому краю, центру, правому краю, ширине)
  • Маркированный и нумерованный списки
  • Tx — очистка форматирования

23. Работа с изображениями

Главное изображение (обложка)

В форме размещения поле Изображение позволяет загрузить одно фото:

  • Нажмите на область загрузки или перетащите файл
  • Форматы: JPG, PNG, WebP
  • Рекомендуемый размер: не менее 1200×800px

Галерея (массовая загрузка)

Поле Галерея (массовая загрузка + crop) позволяет:

  • Загрузить несколько фотографий одновременно
  • Кадрировать (crop) изображения прямо в админке
  • Задать порядок отображения фото

24. FAQ — частые вопросы

Как скрыть объект с сайта, но не удалить его?
Откройте объект на редактирование (Размещения → ✏️) и снимите галочку Активно. Объект исчезнет из каталога, но останется в базе данных.
Почему новое бронирование не отображается на сайте?
Проверьте статус бронирования. Только брони со статусом confirmed считаются подтвержденными и отображаются в личном кабинете клиента.
Как добавить фото в существующий объект?
Перейдите в Размещения → откройте объект на редактирование → в поле Галерея загрузите новые фотографии → сохраните.
Что делать, если платеж не прошел?
Проверьте раздел Оплаты → События платежей. Там будет указана причина ошибки (например, недостаточно средств, отклонен банком). При системной ошибке обратитесь к разработчику.
Как изменить email, с которого отправляются письма?
Перейдите в Почта → SMTP настройки. Измените поле "From Email". Убедитесь, что новый адрес подтвержден в настройках почтового сервера.
Можно ли массово изменить цены?
Да — через Прайс-лист. Подготовьте Excel-файл с новыми ценами и загрузите его. Система автоматически обновит JSON-данные.
Что означает "Показывать на главной"?
Объект с этой галочкой отображается в блоке "Рекомендуем" на главной странице сайта. Используйте для продвижения популярных номеров.
Как удалить администратора?
Перейдите в Администраторы, найдите пользователя и нажмите 🗑. Учетная запись будет удалена безвозвратно. Для временного отключения снимите галочку "Активен".
