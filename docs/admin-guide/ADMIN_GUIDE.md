## 4. Быстрый старт

### Как создать новое размещение с нуля

1. Перейдите в раздел **Типы размещения** → создайте категорию (например, "Домик").
2. Перейдите в раздел **Размещения** → нажмите **"+ Добавить Accommodation"**.
3. Заполните поля: Название, Описание, Тип размещения, Вместимость, Цена за ночь.
4. Загрузите главное изображение (обложку) и галерею фотографий.
5. Установите галочки **Активно** и при необходимости **Показывать на главной**.
6. Нажмите **Сохранить**.

### Как обработать новое бронирование

1. Раздел **Бронирование** — новая бронь появляется со статусом `pending`.
2. Откройте бронь на редактирование (иконка ✏️).
3. Проверьте данные клиента, даты, объект.
4. Измените **Статус** на `confirmed` (подтверждено) или `cancelled` (отменено).
5. Добавьте примечания при необходимости.
6. Сохраните.

### Как обновить прайс-лист

1. Подготовьте Excel-файл с актуальными ценами.
2. Перейдите в **Прайс-лист** → откройте запись на редактирование.
3. Загрузите новый файл в поле **Excel файл**.
4. Сохраните — система автоматически распарсит данные в JSON.

> **Важно:** Формат Excel-файла должен соответствовать ожидаемой структуре. При изменении структуры прайс-листа свяжитесь с разработчиком.

## 21. Общие правила работы с таблицами

Во всех разделах списков работают следующие функции:

| Функция | Описание |
|---------|----------|
| **Показать N строк** | Выбор количества записей на странице (10, 25, 50, 100) |
| **Поиск** | Быстрый поиск по ключевым полям |
| **Конструктор поиска** | Расширенные фильтры по нескольким полям |
| **Видимость столбцов** | Скрытие/показ колонок |
| **Экспорт** | Выгрузка в CSV/Excel |
| **Пагинация** | Переключение между страницами |

**Иконки действий:** 👁 Просмотр, ✏️ Редактирование, 🗑 Удаление (требует подтверждения).

## 22. Работа с Rich Text (WYSIWYG-редактор)

![Rich Text Editor](parkrelax_admin_15_form_accommodation_type.png)

Поля "Описание" используют визуальный редактор:

- **Отменить / Повторить** — стрелки влево/вправо
- **B** — жирный текст
- *I* — курсив
- **Подчеркивание** — маркер/подчеркивание
- **Выравнивание** — по левому краю, центру, правому краю, ширине
- **Списки** — маркированный и нумерованный
- **Tx** — очистка форматирования

## 24. FAQ

**Как скрыть объект с сайта, но не удалить его?**  
Откройте объект на редактирование (Размещения → ✏️) и снимите галочку **Активно**. Объект исчезнет из каталога, но останется в базе данных.

**Почему новое бронирование не отображается на сайте?**  
Проверьте статус бронирования. Только брони со статусом `confirmed` считаются подтвержденными и отображаются в личном кабинете клиента.

**Как добавить фото в существующий объект?**  
Перейдите в **Размещения** → откройте объект на редактирование → в поле **Галерея** загрузите новые фотографии → сохраните.

**Что делать, если платеж не прошел?**  
Проверьте раздел **Оплаты → События платежей**. Там будет указана причина ошибки. При системной ошибке обратитесь к разработчику.

**Как изменить email, с которого отправляются письма?**  
Перейдите в **Почта → SMTP настройки**. Измените поле "From Email". Убедитесь, что новый адрес подтвержден в настройках почтового сервера.

**Можно ли массово изменить цены?**  
Да — через **Прайс-лист**. Подготовьте Excel-файл с новыми ценами и загрузите его. Система автоматически обновит JSON-данные.

**Что означает "Показывать на главной"?**  
Объект с этой галочкой отображается в блоке "Рекомендуем" на главной странице. Используйте для продвижения популярных номеров.

**Как удалить администратора?**  
Перейдите в **Администраторы**, найдите пользователя и нажмите 🗑. Для временного отключения снимите галочку "Активен".
